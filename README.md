# PlayerMounts
PlayerMounts Plugin for Spigot/Paper 1.16+ by GuitarXpress (HeyImJ0hn)

###### Commands:
- /mounts - shows commands
- /mounts toggle - Toggles PlayerMounts On/Off
- /mounts eject - Ejects current passenger

###### Aliases:
  - plym
  - playermounts

###### Permission node:
- "plym.use"
